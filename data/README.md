# Data

This folder contains a small demo QSAR dataset for running the public workflow.

## Files

- `sample_qsar_dataset.csv` - demo compound table with SMILES, IC50 values in micromolar, and calculated pIC50.
- `new_compounds.csv` - example compounds for prediction.

## Important note

The original client/confidential dataset is not included. This public repository uses a small demonstration dataset so the workflow structure can be reviewed and executed safely.
